# Git Шпаргалка

## Инициализация проекта
  ```bash
  git init                     # Инициализировать Git репозиторий
  git clone <url>              # Скопировать чейто репозиторий в текущую папку
  
##Настройка пользователя
  git config --global user.name "Имя"
  git config --global user.email "почта@example.com"
  
#Подключение к репозиторию на гитхабе
  git remote add origin https://github.com/MayaviOff/Notes.git
  origin -- имя хранилища (их несколько может быть но обычно одно)
  https://github.com/MayaviOff/Notes.git -- ссылка на репозиторий
  
#Добавление изменений
  git status                  # Проверить статус изменений
  git add <файл>              # Добавить файл в индекс
  git add .                   # Добавить все файлы

#Комиты
  git commit -m "Коментар"    # Сделать комит с сообщением
  
#Ветки
  git branch                 # Показать все ветки
  git branch <назва>         # Создать новую ветку
  git checkout <назва>       # Перейти к ветке
  git checkout -b <назва>    # Создать и перейти
  
#Обновления и пуш файлов
  git pull origin main        # Загрузить изменения
  git push origin main        # Отправить изменения

#Откат
  git restore <файл>          # Вернуть файл к последнему комиту
  git reset HEAD <файл>       # Убрать из индексу (оставить файл)
  git reset --hard            # Полностью откатить все локальные изменения
  
#Короткая подсказка
  Для комита надо ввести пароль и ник. Пароль - ключ аутетнификации из гитхаба (надо создавать)
