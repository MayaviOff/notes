nslookup — утилита для получения DNS-записей доменов. Дает возможность узнать IP-адрес сайта, MX-записи, NS-записи и другое.

# Узнать IP-адрес домена:
nslookup example.com

# Использовать конкретный DNS-сервер:
nslookup example.com 8.8.8.8

# Получить только MX-записи (почтовые сервера):
nslookup -query=mx example.com

# Получить только NS-записи (какие DNS-сервера обслуживают домен):
nslookup -query=ns example.com

# В папке лежит скриптик для автоматизации
# Для сохранения в файлик надо прописать nslookup... > fileDirectory
